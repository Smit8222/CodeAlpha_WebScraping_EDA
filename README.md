# CodeAlpha_WebScraping_EDA
This project focuses on extracting book data from a public website using Python and BeautifulSoup, followed by data cleaning, exploratory data analysis (EDA), and visualization using Pandas, Matplotlib, and Seaborn. The goal is to transform raw web data into meaningful insights.
# 📊 CodeAlpha Data Analytics Internship Project

## 🔍 Project Title:
Web Scraping and Exploratory Data Analysis of Book Dataset

## 📌 Description:
This project involves extracting book data from a website using Python and performing Exploratory Data Analysis (EDA) with visualization.

## 🧰 Tools Used:
- Python
- BeautifulSoup
- Pandas
- Matplotlib
- Seaborn

## 📊 Tasks Completed:
✅ Web Scraping  
✅ Data Cleaning  
✅ Exploratory Data Analysis  
✅ Data Visualization  

## 📁 Files:
- scraper.py → Extracts data
- analysis.py → Performs EDA & visualization
- books_data.csv → Dataset

## 📈 Output:
- Dataset created from website
- Price distribution graph
- Availability analysis

## 🎯 Conclusion:
The project demonstrates how raw web data can be transformed into meaningful insights using data analysis techniques.
